A Top Signals
1腾讯云 ADP 模型涨价今日零点正式生效：Agent 推理成本进入商业化定价时代
来源： 腾讯云官方公告 / 东方财富网 | 同花顺 （公告 2026-03-11，生效 2026-03-13 00:00 CST）
3月13日零点，腾讯云智能体开发平台（Tencent Cloud ADP）涨价正式落地。GLM 5、MiniMax 2.5、Kimi 2.5 三个第三方模型结束免费公测，按量商用计费；混元 HY2.0 Instruct 输入价格从 0.0008 元/千Token 上调至 0.004505 元（涨幅 463%），输出价格从 0.002 元上调至 0.01113 元（涨幅 456%）。套餐订阅用户可通过套餐抵扣部分费用。
数据平台映射：涨价生效日意味着所有基于腾讯云 ADP 的 Agent 应用从今天起面临真实成本压力。Agent 模式下单任务 Token 消耗较 Chatbot 高 2-3 个数量级，叠加数倍涨价，企业必须立即建立 Token 消耗监控、成本归因和 FinOps 能力。数据平台需要提供推理成本可观测性——哪些 Agent 任务消耗最多 Token、哪些可以通过缓存复用降低成本。
2腾讯推出 OpenClaw 安全工具箱：覆盖云端、企业本地、个人电脑三大场景
来源： 腾讯安全 / IT之家 | 钛媒体 | 36氪 （2026-03-12）
3月12日，腾讯正式推出 OpenClaw 安全工具箱，三层架构覆盖全场景：(1) 云端——Lighthouse 轻量服务器及 ClawPro 企业版提供"物理防爆箱"架构，AI Agent 安全中心统一管理云上 Agent，监测异常指令、扫描技能风险；(2) 企业本地——腾讯 iOA 构建六层自动化防护链路；(3) 个人电脑——腾讯电脑管家 18.0 新增"龙虾管家-AI 安全沙箱"。此外，安全能力被封装为 AI Skills 上架 ClawHub：EdgeOne ClawScan 支持一句话安全体检，HaS Anonymizer 可识别 7 万种实体、脱敏 21 种敏感对象。
数据平台映射：HaS Anonymizer 提供的本地化 PII 脱敏能力（7 万种实体识别、21 种敏感对象图片脱敏），直接填补了 Agent 工作流中的数据隐私治理空白。数据平台如果能与这类安全 Skills 原生集成，将获得 Agent 时代数据治理的先发优势。
3火山引擎 ArkClaw 升级 AI 助手全链路安全方案：沙箱隔离 + 最小授权 + 审计日志
来源： 火山引擎 / 新京报 | 东方财富网 | CITNews （2026-03-12）
火山引擎宣布 ArkClaw 全面升级安全解决方案，构建"事前-事中-事后"三层纵深防御：(1) 事前——提示词意图识别自动拦截高危指令；(2) 事中——沙箱隔离运行，所有可疑网络请求和系统调用实时监控并叫停；(3) 事后——完整操作轨迹形成不可篡改的审计日志。实施"最小授权"与"显式授权"机制。
数据平台映射：火山引擎的方案将 Agent 安全治理拆解为数据平台可直接映射的三大能力：运行时隔离（对应数据沙箱）、细粒度权限控制（对应最小权限数据访问）、不可篡改审计日志（对应数据操作溯源）。这三项能力正在从"安全加分项"变为"平台必备项"。

2京东云 OpenClaw 云服务用户规模单周增长超 300%
来源： 京东云 / 新浪科技 | 新浪财经 （2026-03-12/13）
京东云披露，自1月官宣上线 OpenClaw 云服务（Clawdbot）后，用户规模单周增长超 300%。3月13日在京东集团北京总部开展 OpenClaw 免费安装活动，提供百万 Tokens 免费使用。
影响判断：Agent 云服务的爆发式增长验证了 Agent 运行正在从本地向云端迁移。Agent 产生的数据集中到云端，数据平台的云端日志处理和存储能力面临弹性扩容压力。
3OpenClaw 安全风险持续升级：全球 23 万+实例公网暴露，供应链攻击矩阵形成
来源： Declawed.io / 东方财富网 | 启明星辰 / 搜狐 | Oasis Security / 搜狐 （2026-03-12，综合多日数据）
据 Declawed 最新数据，全球超过 23 万台 OpenClaw 实例暴露于公网。攻击矩阵包括 NPM 恶意依赖包投毒、伪造 GitHub 仓库、ClawJacked 零点击漏洞（CVSS 8.0+）。约 30% 实例未启用认证，约 60% 存在凭证泄露。APT37、APT28 等国家级攻击者正在利用暴露节点。
影响判断：Agent 安全问题已从"潜在风险"升级为"活跃攻击"。数据平台必须在 Agent 数据访问层面提供零信任架构——默认不信任、逐次授权、全程审计。

来源： AWE 2026 现场 / 搜狐 | 东方财富网 （2026-03-12/13）
AWE 2026 在上海开幕。MiniMax 展区重点展示 MiniMax Agent 和 MaxClaw 云端 AI 助手，支持跨端联动。MiniMax 市值 3826 亿港元已超越百度。
影响判断：Agent 从线上走到消费电子展，标志着 Agent 正在渗透消费场景。消费级 Agent 的普及将产生大量用户行为数据，数据平台需要支撑用户画像和行为分析数据流。
C People & Views
1腾讯安全负责人：AI Agent 的高级权限使其成为攻击者的"瑞士军刀"
来源： 腾讯安全 / 新浪网 （2026-03-12）
腾讯相关负责人表示，AI Agent 拥有执行系统命令、读写文件、调用插件等高级权限。一旦被攻击者利用，可能导致凭证密钥被窃取、浏览器被劫持、加密资产被盗。金融、政务、医疗等行业面临最大风险。
数据平台映射：Agent 的"能力即风险"特性要求数据平台从"应用级授权"转向"任务级授权"，每一次数据操作都需要独立的权限验证和审计记录。
2启明星辰：针对 OpenClaw 的常态化、低门槛渗透能力已形成
来源： 启明星辰 / 新浪网 （2026-03-12）
启明星辰指出，针对 OpenClaw 的攻击已形成"全链条攻击矩阵"——从供应链投毒到认证逻辑缺陷渗透，攻击门槛持续降低。每一台被攻陷的 Agent 主机都可能成为深入企业内网的跳板。
数据平台映射：Agent 作为企业内网的"合法居民"，一旦被攻陷将绕过所有网络边界防护直达数据层。数据平台需要将 Agent 视为"半可信客户端"，在数据访问接口实施独立的身份验证和行为异常检测。
