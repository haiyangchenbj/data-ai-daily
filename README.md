## Sections

| # | Section | Focus |
|---|---------|-------|
| A | Top Signals | Breaking news, major releases, policy changes |
| B | Product & Tech | New features, architecture, benchmarks |

| D | Capital & Corporate | Funding, M&A, earnings, partnerships |

## Architecture

- **Engine**: v4.2 - AI-powered search, write, review pipeline
- **Channels**: WeChat Work (企微) + GitHub Pages
- **Schedule**: Mon-Fri, auto-generated 09:30, published 10:00 (UTC+8)
- **Archive**: `/archive/` directory stores historical issues

## Repository Structure

```
index.html          # Latest issue (GitHub Pages entry)
archive/            # Historical daily briefs (HTML)
monthly/            # Monthly digest pages
```
