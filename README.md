## ⚙️ 系统架构

- **引擎版本**: v4.2
- **工作流**: AI 搜索 → 撰写 → 审核 → 发布
- **推送渠道**: 企业微信 + GitHub Pages
- **发布时间**: 周一至周五，09:30 生成，10:00 推送 (UTC+8)
- **归档**: 历史日报存储在 `archive/` 目录，月报存储在 `monthly/` 目录

## 📁 仓库结构

```
index.html      # 最新一期日报 (GitHub Pages 入口)
archive/        # 历史日报归档 (HTML)
monthly/        # 月度汇总报告 (HTML)
README.md       # 本文件
```
