Top Signals

1. Microsoft FabCon+SQLCon 2026：Database Hub 统一管理六大数据库，Fabric IQ 为 AI Agent 构建语义层

来源：Azure Blog — FabCon and SQLCon 2026
微软在 FabCon 2026 发布 Database Hub（Early Access），将 Azure SQL、Cosmos DB、PostgreSQL、MySQL、SQL Server 和 Fabric SQL 纳入统一管理视图，内置 Copilot 智能运维代理，并推出数据库节省计划最高省 35%。同时发布 Fabric IQ 语义层及 MCP Server 预览，使 AI Agent 可通过语义模型发现和操作业务实体。OneLake 新增 Databricks Unity Catalog 读取（公开预览）和 Snowflake 互操作（GA）。
💡 微软正在将 Fabric 从分析平台升级为统一的数据控制面——Database Hub 意味着企业不再需要在多个数据库管理界面间切换，Fabric IQ + MCP 则为多 Agent 系统提供了标准化的数据语义接口。

2. Snowflake 发布 Project SnowWork：自主 AI 平台让业务用户直接从数据获取执行结果

来源：Snowflake Press Release
Snowflake 发布 Project SnowWork 研究预览版，这是一个面向业务用户的自主 AI 平台。不同于传统的问答式 AI 助手，SnowWork 能自主规划并执行多步骤工作流——从数据查询、分析到生成 board-ready 幻灯片或流失风险报表，全程在 Snowflake 安全治理边界内完成。平台内置角色感知的预设技能（财务、销售、运营等），自动继承 RBAC 权限和审计日志。
💡 Snowflake 正从「洞察平台」向「行动平台」转型。SnowWork 的核心命题是让数据平台的价值不再止步于 Dashboard，而是直接交付业务结果。

3. Azure Databricks FabCon 2026 三项重磅发布：Lakeflow Connect 免费层 + Genie Code + Lakebase GA

来源：Databricks Blog — What's New at FabCon 2026
Databricks 在 FabCon 2026 宣布三项重要更新：（1）Lakeflow Connect 免费层——每个工作区每天 100 个免费 DBU，可接入约 1 亿条记录，支持 9 大数据库和主流 SaaS 应用，号称帮助团队构建管道速度提升 25 倍、ETL 成本降低 83%；（2）Genie Code——AI Agent 模式的数据工程助手，可通过自然语言构建管道、调试 Lakeflow 故障；（3）Lakebase GA——基于 Postgres 的无服务器数据库，专为 AI Agent 状态管理设计，已在 14 个 Azure 区域上线。
💡 免费数据接入层是典型的「先圈数据再卖计算」策略，直接降低了从竞品（Snowflake/Redshift/BigQuery）迁移数据的门槛。Genie Code 则标志着数据工程师的工作方式正在被 AI Agent 重新定义。
